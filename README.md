# [veradas-terkep.hu](https://veradas-terkep.hu)

Vibe coded blood donation map of Hungary, which can:
- Visualize locations on a map
- Notify users if there is an upcoming appointment in their area
- Apply filters in the time domain
- Provide all this without mobile app or registration
